::: tip 简介

使用Octoprint与树莓派实现网络控制3D打印机，结合内网穿透实现外网远程监控3D打印机。

:::

<!-- more -->

## 硬件准备

::: tip 树莓派3B一块

以及配套的电源线、连接打印机的usb线

（最好准备一根网线，有线连接可以省去很多麻烦）

:::



::: tip 8G或以上Tf卡一张

以及读卡器一个

用于安装系统

:::
